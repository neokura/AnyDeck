# Security Policy

AnyDeck is a root Decky plugin, so security issues matter.

## Reporting

Please do not open a public issue first for a security-sensitive bug.

Report it privately through GitHub security reporting for the repository if available, or contact the maintainer directly through GitHub.

## Scope

Security-relevant issues include:

- privilege escalation paths
- unsafe command execution
- unsafe managed-file writes
- shell injection opportunities
- incorrect trust of device or runtime input

## Disclosure

After a fix is available, coordinated public disclosure is welcome.
